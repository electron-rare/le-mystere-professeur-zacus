# Prompt : carte objet / indice

- Format : A6, orientation portrait, titre « Objet » et subtitre « Indice ».
- Visual : montre un objet (stylo, partition, gants) avec des lignes de note autour.
- Ajoute une section « Effet » qui décrit le progrès de l’enquête (ex. « Déclenche l’audio de la station suivante »).
- Inclure un mini-tableau « Confirmé par » pour noter les preuves.
