# UML UI RP2040 TFT

Touch UI based on LVGL and TFT_eSPI.

## Main structure

```
main.cpp
  - TFT_eSPI
  - XPT2046 touch
  - UiLinkClient (UART1 GP0/GP1, 57600)
  - LVGL widgets
```

## Key classes

```
UiLinkClient
  + begin
  + poll
  + sendHello
  + sendButton
  + connected

LVGL port
  + lvgl_port_init
  + lvgl_port_disp_flush
  + lvgl_port_touchpad_read
```

## Notes

- Widgets show link, mode, track, volume, tune, uptime.
- Current firmware sends BTN events only (touch is not wired to UI Link yet).
