# UML UI ESP8266 OLED

Lightweight UI firmware for SSD1306 OLED (128x64).

## Main structure

```
main.cpp
  - SoftwareSerial UI link (D6 RX, D5 TX, 57600)
  - Adafruit_SSD1306 display
  - TelemetryState
  - LinkMonitor
  - RenderScheduler
```

## Key classes

```
TelemetryState
  + mode
  + track / trackTotal
  + volume
  + tuningOffset
  + startup
  + app
  + ui_page
  + seq / uptime

LinkMonitor
  + update(nowMs, rxOccurred)
  + isAlive / isDown
  + consecutiveLoss

RenderScheduler
  + shouldRender(nowMs, stateDirty)

StatParser
  + parseFrame(line, state)
```

## Notes

- UI renders are rate-limited (default 250ms).
- Link down displays an explicit warning screen.
