
# Boîtier / Enclosure

## Media Kit Freenove

- **Type** : Boîtier plastique multi-usage (Media Kit)
- **Référence** : Freenove Media Kit Box (ex : FNK0052 ou équivalent)
- **Dimensions** : ~ 120 x 80 x 35 mm (vérifier selon version)
- **Matériau** : ABS noir, couvercle transparent
- **Ports** :
	- 2 ouvertures latérales (USB, alimentation)
	- 1 ouverture frontale (audio, boutons)
	- 1 grille pour ventilation
- **Fixation** : inserts pour vis M2/M3, pads antidérapants
- **Utilisation** :
	- Accueille carte ESP32, UI, alimentation, HP, boutons
	- Prédécoupes pour connecteurs (USB, jack, etc.)
- **Lien officiel** : [Freenove Media Kit](https://www.freenove.com/media-kit.html)
