# Open Source Bill of Materials (ESP32 firmware)

Ce firmware est distribue sous `GPL-3.0-or-later`.

Dependances principales:

- `ESP8266Audio` (`earlephilhower/ESP8266Audio`) - `GPL-3.0`
- `arduino-audio-driver` (`pschatzmann/arduino-audio-driver`) - `GPL-3.0`
- `Adafruit SSD1306` - `BSD-3-Clause`
- `Adafruit GFX Library` - `BSD-2-Clause`
- `EspSoftwareSerial` - `LGPL-2.1`

Notes:

- Toutes les dependances listees sont open source.
- Les sous-dependances transitives conservent leurs licences d'origine.
- Toujours verifier les licences exactes sur les depots amont avant redistribution commerciale.
