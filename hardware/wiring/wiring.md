
# Câblage

## Exemple : Media Kit Freenove

- **ESP32 Audio Kit** fixé sur entretoises, connecteurs accessibles via ouvertures latérales
- **UI (OLED ou TFT)** montée sur façade ou couvercle
- **HP 8Ω** fixé sur grille latérale, câblé sur bornier
- **Boutons poussoirs** sur façade, câblés sur GPIO ESP32
- **Alimentation 5V** via port USB ou bornier

**Notes sécurité** :
- Vérifier la polarité de l’alimentation
- Isoler les fils audio et HP
- Utiliser des entretoises pour éviter les courts-circuits
