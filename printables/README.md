# Printables

Organisation :
- `src/` = sources éditables
- `export/pdf/` = prêts à imprimer
- `export/png/` = previews

Convention de nommage recommandée :
- `a6` / `a5` / `a4`
- `recto` / `verso`
- `v1`, `v2`, etc.

Exemples :
- `invite-a6-recto-v1.pdf`
- `carte-indice-a6-v2.pdf`
- `regles-1-page-a4-v1.pdf`

Ordre d’impression : voir `ordre-dimpression.md`.
